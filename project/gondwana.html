# Gondwana

Gondwana 在地质学中，冈瓦纳大陆（英语：Gondwana，/ɡɒndˈwɑːnə/，或Gondwanaland），也称冈瓦那大陆、冈瓦纳古陆、南方大陆，是存在于新元古代至侏罗纪前期（约5.73亿至1.8亿年前）的超大陆，它是从罗迪尼亚大陆分裂出来的两块超大陆之一（另一个超大陆为劳亚大陆），存在于南半球。
