# Laurasia

劳亚大陆（英语：Laurasia）又译劳拉西亚大陆，为地质史古陆之一，存在于侏罗纪到白垩纪。劳亚大陆包含现今北半球大部分陆地。由劳伦大陆、波罗地大陆、西伯利亚大陆、哈萨克大陆、华北陆块、华南陆块构成，在侏罗纪中期由盘古大陆北端分裂而成，然后在白垩纪分裂成今天的欧亚大陆和北美大陆。其名称是由劳伦大陆（Laurentia）与欧亚大陆（Eurasia）组成。
